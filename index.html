សាលា ប៊ែលធី អន្តរជាតិ | BELTEI International School Website

ឯកសារណែនាំពេញលេញសម្រាប់អ្នកចាប់ផ្តើមរៀន HTML/CSS/JavaScript

១. ទិដ្ឋភាពទូទៅ (Overview)

គេហទំព័រនេះសាងសង់ដោយប្រើអ្វី?

គេហទំព័រនេះគឺជា Single Page Website (គេហទំព័រទំព័រតែមួយ) ដែលប្រើប្រាស់បច្ចេកវិទ្យាចំនួន ៣ គឺ HTML, CSS, និង JavaScript (jQuery)។

HTML

ជា​គ្រោង​សម្រាប់​បង្ហាញ​មាតិកា (អក្សរ, រូបភាព, ប៊ូតុង, ទម្រង់)។ គិតថាវាជា "ខ្លួន" របស់គេហទំព័រ។

CSS

ជា​ស្បែក​សម្រាប់​តុបតែង​គេហទំព័រ (ពណ៌, ទំហំ, ទីតាំង, អានីមេស្សិន)។ គិតថាវាជា "សម្លៀកបំពាក់" របស់គេហទំព័រ។

JavaScript

ជា​ខួរក្បាល​សម្រាប់​ធ្វើឲ្យ​គេហទំព័រ​មាន​ជីវិត (រូបភាពរត់, ម៉ឺនុយបើកបិទ, ប៊ូតុងរអិល)។

jQuery

ជា​បណ្ណាល័យ (Library) ដែលធ្វើឲ្យ JavaScript សរសេរងាយស្រួលជាង (ប្រើសញ្ញា $)។

២. រចនាសម្ព័ន្ធ HTML (HTML Structure)

រចនាសម្ព័ន្ធ HTML គឺជាអ្វី?

HTML ប្រើ Tags (ស្លាក) ដើម្បីបង្ហាញមាតិកា។ ឧទាហរណ៍៖ <div>, <p>, <h1>។ គ្រប់ Tag ត្រូវបើក <tag> ហើយបិទ </tag>។

<!-- ផ្នែកក្បាលទំព័រ --> <head> <title>សាលា ប៊ែលធី អន្តរជាតិ</title> <link rel="stylesheet" href="style.css"> </head> <!-- ផ្នែករាងកាយទំព័រ --> <body> <div class="topbar">...</div> <!-- របារខាងលើ --> <header class="header">...</header> <!-- បឋមកថា --> <nav class="nav">...</nav> <!-- ម៉ឺនុយ --> <section class="hero">...</section> <!-- រូបភាពធំ --> <div class="page-layout">...</div> <!-- មាតិកា + Sidebar --> <footer class="footer">...</footer> <!-- ជើងទំព័រ --> </body>
ពន្យល់ Tags សំខាន់ៗ៖

• <div> = ប្រអប់ (Container) សម្រាប់ដាក់មាតិកា
• <section> = ផ្នែកមួយនៃទំព័រ (ដូចជា Hero, About, News)
• <header> = ផ្នែកក្បាល (Logo, ឈ្មោះសាលា)
• <nav> = ម៉ឺនុយរុករក (Navigation Menu)
• <footer> = ផ្នែកខាងក្រោម (ព័ត៌មានទំនាក់ទំនង)
• <img> = បង្ហាញរូបភាព
• <a> = តំណភ្ជាប់ (Link)

៣. CSS - ការតុបតែង (Styling)

CSS ធ្វើអ្វី?

CSS (Cascading Style Sheets) គឺជាភាសាសម្រាប់តុបតែង HTML។ វាប្រាប់ Browser ថា អ្វីៗត្រូវមាន ពណ៌អ្វី, ទំហំប៉ុន្មាន, និង នៅទីតាំងណា។

/* ឧទាហរណ៍៖ រចនាសម្ព័ន្ធ Top Bar */ .topbar { background: #0a1a3a; /* ពណ៌ផ្ទៃខាងក្រោមខៀវចាស់ */ color: rgba(255,255,255,.85); /* ពណ៌អក្សរស (85% opacity) */ font-size: 13px; /* ទំហំអក្សរ 13px */ border-bottom: 2px solid #f59e0b; /* បន្ទាត់ក្រោមមាស */ } /* ឧទាហរណ៍៖ រចនាសម្ព័ន្ធ Header */ .header { background: #ffffff; /* ពណ៌ផ្ទៃស */ position: sticky; /* បិទជាប់ខាងលើពេលរអិល */ top: 0; /* បិទនៅចំណុចខាងលើទំព័រ */ z-index: 900; /* ស្រទាប់ខ្ពស់ (លើគេ) */ }
ពន្យល់ CSS Properties សំខាន់ៗ៖

• display: flex = រៀបធាតុក្នុងបន្ទាត់តែមួយ (ដូចកៅអី)
• grid-template-columns = បែងចែកជាជួរឈរ (ដូចតារាង)
• position: sticky = បិទជាប់ពេលរអិលទំព័រ
• transition = ធ្វើឲ្យការផ្លាស់ប្តូររលូន (smooth)
• border-radius = ធ្វើឲ្យមុំមូល
• box-shadow = ស្រមោលខាងក្រោម
• hover = រចនាសម្ព័ន្ធពេល Mouse ដាក់លើ

៤. ប្រព័ន្ធបែងចែកទំព័រ (Layout System)

ទំព័រនេះប្រើប្រព័ន្ធ Grid ដើម្បីបែងចែកមាតិកា

គេហទំព័រមាន ២ ផ្នែកធំៗ៖ ផ្នែកឆ្វេង (មាតិកា) និងផ្នែកស្តាំ (Sidebar)។ នេះប្រើ CSS Grid ដើម្បីបែងចែក។

/* ប្រព័ន្ធ Grid សម្រាប់បែងចែកទំព័រ */ .page-layout { display: grid; grid-template-columns: 1fr 300px; /* 1fr = ផ្នែកឆ្វេងយកទំហំដែលនៅសល់ */ /* 300px = ផ្នែកស្តាំទទឹង 300px */ gap: 28px; /* គម្លាតរវាងផ្នែក */ }

ផ្នែកឆ្វេង (Main Content)

មាន Cards ជាច្រើន៖ Programs, About, Extra Programs, Facilities, Competitions, News, Videos។ ផ្នែកនេះទទឹងធំជាង (1fr)។

ផ្នែកស្តាំ (Sidebar)

មាន Side Cards ជាច្រើន៖ បញ្ជីឈ្មោះសិស្សបញ្ចប់, ប្រកួតនិយាយ, ប្រកួតជជែក, ប្រកួតអំណាន។ ទទឹង 300px ជាប់នៅកន្លែង (sticky)។

៥. Components សំខាន់ៗ (សមាសភាគ)

១

Top Bar (របារខាងលើ) — ផ្នែកខ្ពស់បំផុតនៃទំព័រ។ មានទូរសព្ទ, អ៊ីមែល, និងប៊ូតុងបណ្តាញសង្គម (Facebook, YouTube, Telegram, Instagram)។ ពណ៌ខៀវចាស់ #0a1a3a ជាមួយបន្ទាត់មាសក្រោម។

២

Header (បឋមកថា) — មាន Logo សាលា, ឈ្មោះសាលា "សាលា ប៊ែលធី អន្តរជាតិ", និងស្លាក "BELTEI — the best quality of education in Cambodia"។ ប្រើ position: sticky ដើម្បីបិទជាប់ពេលរអិល។

៣

Navigation (ម៉ឺនុយ) — មាន ៨ ម៉ឺនុយធំៗ៖ ទំព័រដើម, អំពីសាលា, កម្មវិធីសិក្សា, កម្មវិធីបន្ថែម, សម្ភារៈបរិក្ខារ, សិស្សពូកែ, ការប្រកួត, ព្រឹត្តិការណ៍, ទំនាក់ទំនង។ មាន Dropdown Menus (ម៉ឺនុយធ្លាក់ចុះ) ពេល Mouse ដាក់លើ។

៤

Hero Slider (រូបភាពរត់) — រូបភាពធំ ៣ សន្លឹកដែលប្តូរគ្នាស្វ័យប្រវត្តិរាល់ ៥ វិនាទី។ មានប៊ូតុងព្រួញឆ្វេង/ស្តាំ, ចំណុច (Dots), និង Counter "1 / 3"។ មាន Gradient Overlay ដើម្បីធ្វើឲ្យអក្សរអានបាន។

៥

Stats Strip (របារលេខ) — របារខៀវមានលេខ ៤ ចំនួន៖ 32 ទីតាំង, ៦ កម្មវិធី, 50K+ សិស្ស, 20+ ឆ្នាំបទពិសោធន៍។ ប្រើ grid-template-columns: repeat(4, 1fr)។

៦

Cards (ប័ណ្ណ) — ប្រអប់សម្រាប់បង្ហាញមាតិកា។ មាន Header ពណ៌ខៀវចាស់ + បន្ទាត់មាសក្រោម។ ប្រើ border-radius: 18px ដើម្បីមុំមូល និង box-shadow សម្រាប់ស្រមោល។

៧

Programs Grid (កម្មវិធីសិក្សា) — បង្ហាញកម្មវិធី ៥ ក្នុង Grid ៣ ជួរ។ នីមួយៗមានរូបភាព, ចំណងជើងខ្មែរ, និងអង់គ្លេស។ ពេល Mouse ដាក់លើ (hover) មាន transform: scale(1.03) (ធំឡើង 3%) និងស្រមោលខ្លាំងឡើង។

៨

Campus Grid (ទីតាំង ៣២) — បង្ហាញទីតាំងសាលា ៣២ កន្លែងក្នុង Grid ៤ ជួរ (នៅលើ Desktop)។ មានលេខ ១-៣២, រូបភាព, និងឈ្មោះទីតាំង។

៩

News List (បញ្ជីព័ត៌មាន) — បង្ហាញព័ត៌មាន ៥ ចំណងជើងជាមួយរូបភាព, កាលបរិច្ឆេទ, និងអ្នកសរសេរ (Admin)។ ប្រើ flex ដើម្បីរៀបរូបភាពនិងអក្សរក្នុងបន្ទាត់តែមួយ។

១០

Video Grid (វីឌីអូ) — បង្ហាញវីឌីអូ YouTube ៥ ក្នុង Grid ៣ ជួរ។ មានប៊ូតុង Play ពណ៌មាសនៅកណ្តាល។ ប្រើ aspect-ratio: 16/9 ដើម្បីរក្សាទម្រង់វីឌីអូ។

១១

Footer (ជើងទំព័រ) — មាន ៤ ជួរ៖ អំពីសាលា, តំណភ្ជាប់លឿន, កម្មវិធីសិក្សា, ទំនាក់ទំនង។ ពណ៌ខៀវចាស់ #0a1a3a ជាមួយបន្ទាត់មាសខាងលើ។ មានប៊ូតុងបណ្តាញសង្គម។

៦. JavaScript & jQuery (ខួរក្បាលគេហទំព័រ)

JavaScript ធ្វើឲ្យគេហទំព័រមានជីវិត

គេហទំព័រនេះប្រើ jQuery (បណ្ណាល័យ JavaScript) ដើម្បីសរសេរកូដងាយស្រួលជាង។ សញ្ញា $ គឺជាសញ្ញារបស់ jQuery។

// ============================================================ // ផ្នែក A: HERO SLIDER (រូបភាពរត់) // ============================================================ // ប្រកាសអថេរ (Variables) let currentSlide = 0; // រូបភាពបច្ចុប្បន្ន (ចាប់ផ្តើមពី 0) const totalSlides = 3; // ចំនួនរូបភាពសរុប = 3 // ហៅវិធី showSlide() ដើម្បីបង្ហាញរូបភាព function showSlide(number) { // បើលេខធំជាង 2 (ពីព្រោះចាប់ផ្តើមពី 0) ត្រលប់ទៅ 0 if (number >= totalSlides) number = 0; // បើលេខតូចជាង 0 ទៅរូបភាពចុងក្រោយ if (number < 0) number = totalSlides - 1; currentSlide = number; // លាក់រូបភាពទាំងអស់ $('.hero__slide').removeClass('active'); // បង្ហាញតែរូបភាពបច្ចុប្បន្ន $('.hero__slide').eq(currentSlide).addClass('active'); // ធ្វើដូចគ្នាសម្រាប់ Dots (ចំណុច) $('.dot').removeClass('active'); $('.dot').eq(currentSlide).addClass('active'); // ធ្វើឲ្យ Counter ប្តូរលេខ (1 / 3) $('#slideCurrent').text(currentSlide + 1); }
ពន្យល់ jQuery Methods សំខាន់ៗ៖

• $(selector) = ជ្រើសរើសធាតុ HTML (ដូច document.querySelector)
• .addClass('active') = បន្ថែម Class "active" ឲ្យធាតុ
• .removeClass('active') = យក Class "active" ចេញ
• .eq(index) = ជ្រើសរើសធាតុតាមលេខលំដាប់ (0, 1, 2...)
• .click(function(){...}) = ពេលចុច Mouse ធ្វើអ្វី
• .toggleClass('open') = បិទបើក Class (បើមានយកចេញ, បើគ្មានបន្ថែម)
• setInterval(function, 5000) = រត់វិធីរាល់ ៥ វិនាទី

// ============================================================ // ផ្នែក B: MOBILE NAV (ម៉ឺនុយទូរសព្ទ) // ============================================================ function startMobileNav() { // ពេលចុចប៊ូតុង Hamburger $('#navToggle').click(function() { // Toggle = បើកបើកបើបិទ, បើបិទបើបើក $('#navMenu').toggleClass('open'); }); // ពេលចុច Dropdown លើទូរសព្ទ $('.dropdown > a').click(function(e) { if ($(window).width() <= 768) { e.preventDefault(); // កុំឲ្យទៅ Link $(this).closest('li').toggleClass('open'); } }); }

៩. គោលគំនិតសំខាន់ៗ (Key Concepts)

Box Model

គ្រប់ធាតុ HTML គឺជា "ប្រអប់"។ មាន ៤ ផ្នែក៖ Content (មាតិកា) → Padding (គម្លាតខាងក្នុង) → Border (បន្ទាត់) → Margin (គម្លាតខាងក្រៅ)។

Flexbox

ប្រព័ន្ធរៀបធាតុក្នុងបន្ទាត់តែមួយ (ដូចកៅអី)។ ប្រើ display: flex ដើម្បីរៀបធាតុឲ្យស្មើគ្នា ឬពាក់កណ្តាល។

CSS Grid

ប្រព័ន្ធរៀបធាតុក្នុងតារាង (ជួរឈរ + ជួរដេក)។ ប្រើ display: grid និង grid-template-columns។

CSS Animations

ប្រើ @keyframes ដើម្បីបង្កើតចលនា (ដូចជា Dropdown ធ្លាក់ចុះ)។ ប្រើ transition ដើម្បីធ្វើឲ្យការផ្លាស់ប្តូររលូន។

Sticky Position

ធាតុបិទជាប់នៅកន្លែងមួយពេលរអិលទំព័រ។ ប្រើ position: sticky; top: 0 ដើម្បី Header បិទជាប់ខាងលើ។

CSS Variables

ប្រើពណ៌ដដែលគ្រប់ទីកន្លែង (ដូចជា #0a1a3a, #f59e0b, #2563eb)។ ប្រសិនបើចង់ប្តូរពណ៌ គ្រាន់តែប្តូរកន្លែងតែមួយ។

១០. របៀបកែប្រែកូដ (How to Edit)

១

បើកឯកសារ — ប្រើកម្មវិធីដូចជា VS Code, Notepad++, ឬ Sublime Text។ កុំប្រើ Notepad ធម្មតា (មិនមានពណ៌កូដ)។

២

កែអក្សរ — ស្វែងរកអក្សរដែលចង់កែក្នុង HTML (ឧទ៖ "សាលា ប៊ែលធី")។ ផ្លាស់ប្តូរអក្សរហើយ Save (Ctrl+S)។

៣

កែពណ៌ — ស្វែងរកកូដពណ៌ក្នុង CSS (ឧទ៖ #0a1a3a គឺជាពណ៌ខៀវចាស់)។ ផ្លាស់ប្តូរហើយ Save។

៤

ប្តូររូបភាព — ដាក់រូបភាពថ្មីក្នុងថត pic/ ហើយកែ src ក្នុង HTML (ឧទ៖ src="../pic/image-new.png")។

៥

បន្ថែមម៉ឺនុយ — ចម្លង <li> ដែលមានស្រាប់ ហើយបិទបញ្ចូលក្នុង <ul class="nav__menu">។

៦

បង្ហាញលើ Browser — ចុចពីរដងលើ index.html ឬ បើក Chrome ហើយដាក់ File → Open File។

១១. កំហុសទូទៅ (Common Errors)

!

រូបភាពមិនបង្ហាញ — ពិនិត្យ src ថាត្រឹមត្រូវឬអត់ (ឧទ៖ ../pic/image.png vs ./pic/image.png)។ ពិនិត្យថាឈ្មោះឯកសារត្រឹមត្រូវ (ធំតូច)។

!

CSS មិនដំណើរការ — ពិនិត្យថា Class Name សរសេរត្រឹមត្រូវ (ឧទ៖ topbar vs top-bar)។ ពិនិត្យ Brackets {} បិទបើកត្រឹមត្រូវ។

!

JavaScript មិនដំណើរការ — ពិនិត្យ Console (F12 → Console) មើល Error។ ពិនិត្យថា jQuery CDN ដំណើរការឬអត់។

!

Responsive មិនដំណើរការ — ពិនិត្យ @media queries ថាសរសេរត្រឹមត្រូវ។ ប្រើ DevTools (F12) ដើម្បីប្តូរទំហំអេក្រង់។

១២. សង្ខេប (Summary)

អ្វីដែលអ្នកបានរៀន៖

• HTML = គ្រោងគេហទំព័រ (Tags, Elements)
• CSS = តុបតែង (Colors, Layout, Animations)
• JavaScript/jQuery = ធ្វើឲ្យមានជីវិត (Slider, Menu, Scroll)
• Responsive = ឆ្លើយតបទៅនឹងឧបករណ៍ (Desktop, Tablet, Mobile)
• File Structure = រៀបចំឯកសារឲ្យមានលំដាប់លំដោយ

គន្លឹះសំខាន់៖

• រៀបចំកូដឲ្យមានលំដាប់លំដោយ (Comment, ឈ្មោះ Class ច្បាស់)
• ប្រើ Class Name ដែលមានន័យ (ឧទ៖ topbar, header, nav)
• ពិនិត្យកូដរៀងរាល់ពេល Save (Ctrl+S)
• ប្រើ DevTools (F12) ដើម្បី Debug
• រៀនពីកំហុស (Errors ជាគ្រូល្អបំផុត)
